# computer-science-lab

Exploring computer science related topics.

Knowing more about the moving parts underneath has always fascinated me. I have however always struggled with motivating myself to learn theoretical topics in advance of having a practice use for them. My personal method has mostly been to dive deeper on a need-to-basis. This will however be an attempt to poke at a few topics in the more traditional educational style where I in advance don't know if I will be needing the deeper knowledge or not.

The topics till be cherry picked based on my own mood and motivation to keep the interest up.

I'm using C for some of these experiments as it seems to be the recommended language to get close to the hardware while still staying in the realm of using "english" as the programming language. I don't have any plans of using C beyond that which is why you might find me doing a memory-unsafe things in these experiments. I realized early when working with C that it will take a lot of specalization in C in order to develop in a memory-safe way.
